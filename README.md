# FAS-Backend

## 1. Install packages
```
npm install
```

## 2. Create .env and copy contents from .env.sample
```
.env.sample
```

## 3. Run the app
```
npm run dev
```

## 4. Run the test
```
npm run test
```

## 5. Swagger URL
```
https://localhost:3000/api-docs
```

## 6. API URL:
```
http://localhost:3000/api/bookings
```
